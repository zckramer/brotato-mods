[h1]ModOptions - Configuration Framework[/h1]

A powerful and flexible configuration framework for Brotato mods. ModOptions provides an easy-to-use API for mod developers to add in-game configuration interfaces, with all mod settings unified in one convenient "Mods" tab.

[h2]Features[/h2]

[b]For Players:[/b]
[list]
[*][b]Unified Interface[/b] - All mod configurations in one "Mods" tab in Options
[*][b]Live Updates[/b] - Most changes apply immediately without restarting
[*][b]Persistent Settings[/b] - All settings automatically saved and restored
[*][b]User-Friendly[/b] - Clean, consistent interface for all mod settings
[/list]

[b]For Mod Developers:[/b]
[list]
[*][b]Simple API[/b] - Register options with just a few lines of code
[*][b]Rich Option Types[/b] - Sliders, toggles, dropdowns, text inputs, item selectors
[*][b]Zero Boilerplate[/b] - No UI code needed - just define your options
[*][b]Automatic Persistence[/b] - Settings saved and loaded automatically
[*][b]Translation Support[/b] - Full internationalization support
[/list]
